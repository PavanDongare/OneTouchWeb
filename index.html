Frequently Asked Questions

Can we install new Automation panel in place of old Conventional switch board?

Yes, we can install automation panel easily in place of old Conventional switch board. As the installation procedure is same as conventional switch board & Size of new panel is also as per standard size.

Maximum load taken each device on by panel?

Its safe to take up to 5 Amp for each Device connected.

Is Automation free?

Yes the basic features of the Automation are free. The Special features will be displayed in app purchase which are optional.

What if Touch Related issue accours?

Just press rest/home(red) button so that system will reconfigure and your maximum problems will be solved. NOTE: After pressing rest/home(red) button do not touch panel wait till red light is on. If still the problem occurs kindly contact customerCare Support for further assists.

What if I have to change my Home Wi-Fi username/password?

To change the router wifi detail, customer(admin) must change the WiFi details in the mobile App first. Kindly Refer installation video for the process. After the App details changes the customer can change their home Router Details.

What if fan related issues occur?

To overcome this Fan related problems, reduce the fan speedto level as : 3-1-off . Repeat the procedure till fan operation comes to normal or turns Off. Press rest button to reconfigure the device. (The problem in the Fan may accurs due to improper connection or current fluctuations. The panel is absolute safe to use.)

Can I use the device with no Automation?

Absalutely Yes! The product features for day today life working like touch,regulation. rest ,etc does not require automation active.

What if the wifi connection or Server Calls or Mobile Application goes off ?

In the cases of Wifi Connection or Server or mobile App server goes down ; the system is still usable for the normal touch features. In case of the wifi Router restart after long time then the device might take reset once for the connection purpose. The product comes normal ofter one reset.
